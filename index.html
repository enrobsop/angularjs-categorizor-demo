Angularjs-categorizor-demo

A demo application written in AngularJS that clusters and categorises sample bank statement transactions.

Download .zip Download .tar.gz View on GitHub

Welcome

This demonstration...

  1. Identifies clusters of transactions within a test bank statement pre-configured in the AngularJS model.
  2. The clustering strength can be varied and the new clusters are created as the strength changes.
  3. Transactions can be dragged between clusters and clusters can be merged by dragging them onto one another.
  4. Each cluster can then be assigned to a category, for example 'Groceries'.
  5. Categories are updated within the model and new categories are immediately made available to subsequent clusters.

The application integrates AngularJS with the following JavaScript libraries: Masonry, JQueryUI, Flot, Select2 and Bootstrap.
